# CSWeb Reports Prototype

Prototype generating a report from CSWeb's MySQL database

## Usage

### Generate JSON

Convert CSWeb's MySQL database data to geocode and household JSON. The JSON is written out to *geocodes.json* and *households.json* at the web root.

### Display JSON

Display the geocode and household JSON as a table.

### Reports

Using geocode and household JSON display a table that shows the number of synchronized questionnaires by state.
